# 节气花园 - 项目内容总结

## 项目概述
**节气花园**是一款结合中国传统24节气文化的微信小游戏，玩家可以在游戏中体验种植作物、养育宠物、学习农业知识、与其他玩家交流心得，以及与AI进行种植竞技。

---

## 一、技术架构

### 1. 项目类型
- **平台**：微信小游戏
- **开发模式**：云开发（CloudBase）
- **渲染方式**：Canvas 2D
- **编程语言**：JavaScript

### 2. 项目结构
```
Solar Term Garden/
├── game.js              # 游戏入口文件
├── game.json            # 小游戏配置
├── game.wxss            # 样式文件
├── project.config.json  # 项目配置
├── js/
│   └── main.js          # 游戏主逻辑（Canvas渲染）
├── cloudfunctions/      # 云函数
│   ├── chatWithPet/     # AI聊天
│   ├── crowAttack/      # 乌鸦袭击事件
│   ├── createPost/      # 创建帖子
│   ├── addComment/      # 添加评论
│   └── initDatabase/    # 初始化数据库
├── database/            # 数据库集合定义
│   ├── collections/     # 13个集合JSON文件
│   └── IMPORT_GUIDE.md  # 导入指南
└── images/              # 图片资源目录
```

## 二、核心功能模块

### 模块一：首页系统
**功能**：
- 显示当前节气（根据日期自动计算）
- 显示用户金币数量
- 快捷入口导航（花园、宠物、百科、社区、竞技）
- 宠物状态预览
- 花园概览
- 最新种植心得

**技术实现**：
- Canvas绘制UI界面
- 触摸事件检测
- 场景切换系统

---

### 模块二：花园系统
**功能**：
- 9块土地种植
- 4种作物：桃花、西瓜、菊花、菠菜
- 种子选择购买（消耗金币）
- 作物生长系统（10秒成熟）
- 成熟收获（获得金币收益）

**经济系统**：
| 作物 | 种子价格 | 收获价格 | 利润率 |
|------|----------|----------|--------|
| 桃花 | 20💰 | 80💰 | 300% |
| 西瓜 | 30💰 | 120💰 | 300% |
| 菊花 | 25💰 | 100💰 | 300% |
| 菠菜 | 15💰 | 60💰 | 300% |

**技术实现**：
- 地块点击检测
- 生长定时器（setInterval）
- 成熟闪烁动画
- 金币增减计算

---

### 模块三：宠物系统
**功能**：
- 宠物状态管理：饱腹度、精力值、亲密度
- 喂养功能（消耗金币）
- AI聊天互动
- 宠物等级系统

**技术实现**：
- 状态数据存储
- 喂养冷却时间
- AI对话接口（云函数）

---

### 模块四：百科系统
**功能**：
- 植物图鉴（5种植物）
- 病虫害识别（4种病虫害）
- 种植指南
- 节气农事建议

**数据内容**：
- 桃花、西瓜、菊花、菠菜、桂花
- 蚜虫、红蜘蛛、白粉病、烂根
- 24节气完整数据

---

### 模块五：社区系统
**功能**：
- 发布种植心得
- 查看其他玩家帖子
- 点赞评论互动
- 节气标签筛选

**技术实现**：
- 云数据库CRUD
- 图片上传
- 分页加载

---

### 模块六：AI竞技系统
**功能**：
- 与AI比赛种植速度
- 实时进度对比
- 浇水施肥加速
- 胜负判定奖励

## 七、技术亮点

1. **Canvas渲染**：纯代码绘制UI，无需WXML
2. **触摸交互**：完整的点击检测系统
3. **动画效果**：作物成熟闪烁、生长进度
4. **状态管理**：游戏状态机设计
5. **云开发**：无需后端服务器
6. **模块化**：功能模块清晰分离
