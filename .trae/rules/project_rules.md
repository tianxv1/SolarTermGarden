# 节气花园UI/UX设计自动优化规则

## 自动激活条件
当用户请求涉及以下内容时，自动使用UI设计优化：
- 界面设计、UI设计、页面布局
- 样式优化、视觉效果、配色方案
- 组件设计、交互设计

## 行为准则
1. 优先使用软萌治愈国风水彩风格
2. 色彩采用低饱和度、柔和温暖的调色板
3. 字体选择清晰易读的字体，标题可适当使用偏古典风格
4. 所有界面按微信小程序设计规范适配

## 技术栈偏好
- 前端框架：HTML + Tailwind CSS
- 图标库：FontAwesome 6
- 组件库：可参考shadcn/ui的卡片设计
- 响应式：默认支持移动端优先

## 质量要求
- 生成的界面代码具有专业水准
- 包含适当的动画和过渡效果
- 确保所有图标和元素清晰可辨
- 遵循无障碍设计规范
